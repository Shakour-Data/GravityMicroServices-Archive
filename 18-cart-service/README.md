# 18-cart-service

Shopping cart with persistence

## Quick Start

```bash
# Install dependencies
poetry install

# Run service
poetry run uvicorn app.main:app --port 8103 --reload
```

## Port
- **Service Port:** 8103

## Database
- **Type:** Redis+PostgreSQL
