# 08-email-service

Dedicated email service with templates

## Quick Start

```bash
# Install dependencies
poetry install

# Run service
poetry run uvicorn app.main:app --port 8086 --reload
```

## Port
- **Service Port:** 8086

## Database
- **Type:** PostgreSQL
