# 28-chat-service

Real-time messaging with WebSocket

## Quick Start

```bash
# Install dependencies
poetry install

# Run service
poetry run uvicorn app.main:app --port 8120 --reload
```

## Port
- **Service Port:** 8120

## Database
- **Type:** PostgreSQL+Redis
